# Agent Rules — Playbook

## General Rules
- Use English for all code, comments, documentation, and communication
- Answer concisely, straight to the point
- Consistent with existing document format

## Document Types
- **ROLES/**: Role template → standard format with AI instructions
- **WORKFLOWS/**: Procedures → step-by-step format
- **ACCESS/**: Policies → rules + tables format
- **docs/**: Documentation → narrative format with clear headers

## Conventions
- Markdown for all documents
- Relative internal links (../path/file.md)
- Code examples formatted with triple backticks + language
- Tables for structured data
