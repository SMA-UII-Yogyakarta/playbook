# Agent Rules — Playbook

## Aturan Umum
- Gunakan bahasa Indonesia
- Jawab ringkas, langsung ke titik
- Konsisten dengan format dokumen existing

## Jenis Dokumen
- **ROLES/**: Template peran → format baku dengan AI instructions
- **WORKFLOWS/**: Prosedur → format langkah demi langkah
- **ACCESS/**: Kebijakan → format rules + tabel
- **docs/**: Dokumentasi → format naratif dengan header jelas

## Konvensi
- Markdown untuk semua dokumen
- Internal link relatif (../path/file.md)
- Kode example diformat dengan triple backticks + bahasa
- Tabel untuk data terstruktur
